Only tested on Mac OS, with proper dependencies already installed.

:warning: Use at your own risk :warning:


Clone this repository : look ahead for the green button "Clone", a popup should appear, with a "Download ZIP" at the bottom, click on it, once downloaded unzip it.

Download and install Google's Antigravity (you'll figure out how to do it)

Open Antigravity and open the folder you have unzipped.

On the right side you should see a small window where you can chat with Antigravity.
You should see it is in planning mode : change it to fast mode.

In this chat box, type the following : "Follow the instruction of prompt.txt" and press enter.

Accept all the permissions it asks you : beware you are losing control of your computer ;-)
